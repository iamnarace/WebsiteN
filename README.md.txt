# Naresh Adhikari — IT & Systems Support

Professional IT support, computer repairs & smart home automation for Sydney homes and small businesses.

## Live site
https://nareshadhikari.com.au

## Built with
Single-file static site — HTML5, CSS3, vanilla JavaScript. No build step, no dependencies.

## Structure
- `index.html` — the entire website (Home, Services, About, Resume, Contact)
- `assets/images/` — profile photos
- `favicon.svg` — site icon

## Deploy
Served via GitHub Pages. Push to `main` and it's live.
